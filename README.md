# Горячие клавиши:
### Копировать:
Ctrl+C
### Вставить:
Ctrl+V
### Вырезать:
Ctrl+X
### Сохранить:
Ctrl+S
### Отмена:
Ctrl+Z
# Работа с git
### Скачать проект:
```bash
git clone <URL>
```
### Учесть все файлы:
```bash
git add.
```
### Сохранить коммит:
```bash
git commit -m "..."
```
### Отправка в облако:
```bash
git push
```
